By 蔡小靚 25 January 2023

上 Hawhow 課程：Python 入門特訓 - 基礎實作到證照攻略。 學習使用 if 迴圏。 if是十分常見的判斷式，你在英文口語中也經常用到：

  • 如果-便

    if condition:
    　　換行縮排後接動作。

  • 如果-便-否則-便

    if condition:
    　　換行縮排後接動作。
    else:
    　　換行縮排後接動作。

  • 另外還有巢狀式結構的用法

這裡提醒自己的是，條件之後要加上 “:"、並且縮排很重要。 這裡就不會一一列出課堂教的東西，一樣把課程上我寫的小作業貼上做為紀錄。

讓使用者輸入2個整數及運算元，然後印出計算結果

這裡我自己加碼了 運算元什麼都不寫的話就合部印出；另外是除法系列的分不可為０判斷也一恲放入。

下面是我的代碼

a=eval(input("Please input 1st integer : "))
b=eval(input("Please input 2nd integer : "))
operator=input("Please choose an operator (+, -, *, /, //, %, **, or all) : ")

if operator=="+" :
    print("加法 : {}{}{} = {}".format(a,operator,b,a+b))
elif operator=="-" :
    print("減法 : {}{}{} = {}".format(a,operator,b,a-b))
elif operator=="*" :
    print("乘法 : {}{}{} = {}".format(a,operator,b,a*b))
elif operator=="/" :
    if b==0:
        print("「除數不可為 0 」")
    else:
        print("除法 : {}{}{} = {}".format(a,operator,b,a/b))
elif operator=="//" :
    if b==0:
        print("「除數不可為 0 」")
    else:
        print("整除 {}{}{} = {}".format(a,operator,b,a//b))
elif operator=="%" :
    if b==0:
        print("「除數不可為 0 」")
    else:
        print("餘數 : {}{}{} = {}".format(a,operator,b,a%b))
elif operator=="**" :
    print("乘方 : {}{}{} = {}".format(a,operator,b,a**b))
else:
    print("加法 : {}+{} = {}".format(a,b,a+b))
    print("減法 : {}-{} = {}".format(a,b,a-b))
    print("乘法 : {}*{} = {}".format(a,b,a*b))
    print("乘方 : {}**{} = {}".format(a,b,a**b))
    if b==0:
        print("「除數不可為 0 」, 無法執行 除法, 整除, 及餘數。")
    else:
        print("除法 : {}/{} = {}".format(a,b,a/b))
        print("整除 : {}//{} = {}".format(a,b,a//b))
        print("餘數 : {}%{} = {}".format(a,b,a%b))

讓使用者輸入西元年份（YYYY），計算該年份是否為閏年後印出

這裡在設計公式時腦子一時錯亂，直接google了閏年公式；直接照他的邏輯寫。

myYear=eval(input("input a year with format YYYY : "))

if myYear%4==0:
    if myYear%100==0:
        if myYear%400==0:
            print("The Year is {} is a Leap Year(366 days)".format(myYear))
        else:
            print("The Year is {} is NOT a Leap Year(365 days)".format(myYear))
    else:
        print("The Year is {} is NOT a Leap Year(365 days)".format(myYear))
else:
    print("The Year is {} is NOT a Leap Year(365 days)".format(myYear))
