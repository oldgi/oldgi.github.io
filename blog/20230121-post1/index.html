How to Preview Markdown text in Visual Studio Code

By 蔡小靚 21 January 2023

我上完了 Hawhow 的課程：Github 免費架站術！輕鬆打造個人品牌。 課程有介一個Typora軟體可以即時Preview Markdown的效果；但我覺得麻煩，不想多裝一個軟體。 在VS Code的extensions中輸入markdown後，找了３個東西安裝。

Extension 1：Markdown All in One

下載人數最多的那一個。不用額外設定裝好就行。讚讚。 裝他最主要是各種自動化功能，像是

＊貼上超連結自動完成語法
＊清單中 Enter 自動接續縮排
＊數學公式等實用功能
＊...

有興趣者可參考 Markdown All in One – Visual Studio Marketplace https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one 的說明內容。

功能多多，可以看文件操作一下

Extension 2：Markdown Preview Enhanced

裝他最重要的就是 Preview。 即使是假掰的微積分公弍也行——使用 \int_積分下限^積分上限 {積分表達式} $$\int_0^1 {x^2} \,{\rm d}x $$ 效果就會是：$$\int_0^1 {x^2} ,{\rm d}x $$ 喜歡公式的人可以參考 如何在 Markdown 輸入數學公式及符號

黃框按下去就可以啟用預覽了

公式的預覽效果

Extension 3：Markdown PDF

故名思意這個自然是能夠讓你將 md file export成pdf。 安裝後就生效了，使用方式也十分容易，在編輯框 left-click-mouse 叫出的浮動視窗中就有 export 成 pdf 的選項。 這個十分容易，我就不截圖了。
