By 蔡小靚 23 January 2023

上 Hawhow 課程：Python 入門特訓 - 基礎實作到證照攻略。 首先不可免俗要在電腦上架設安裝的環境。

安裝 Python

就算課程不教你也可以使用 google search 到 Python 的網站。以今天 (2023-01-23) 來說，他最新的版本是 “3.11.1”。 要特別注意的是，如果你開發要使用的是Windows 7的作業系統環境，那不能安裝3.9之後的版本；要下載時也請到Full version list中去找尋需要的版本。 ‘Note that Python 3.9+ cannot be used on Windows 7 or earlier.’

在裝過桯中，要特別注意，勾選將安裝的路徑加入環境變數中。

以下是我又貪玩問了 chatGPT 如何安裝 Python 的資訊

＊Prompt=“pleases give me instructions step by step before I learn python.”*

以下是我得到的回答：

Sure, here are the steps to install Python on a Windows 11 platform:

  1. Open your web browser and navigate to the official Python website (https://www.python.org/)
  2. Click on the “Downloads” link on the top menu, and then click on the “Windows” button to download the latest version of Python for Windows.
  3. Once the download is complete, open the installer file and follow the prompts to install Python.
  4. During the installation process, make sure to check the option to “Add Python to PATH” or “Add Python to environment variables”, this will allow you to run Python from the command prompt or terminal.
  5. After the installation is complete, open the Command Prompt (press Win+R then type cmd) and type python –version to confirm that Python has been installed successfully. …

我在 windows 11 下安裝 Python 3.11.1 後，python –version 請改用 python -Ｖ 來檢查版本資訊。可以用 python -h 來查有什麼不同的指令；要注意大小寫是不同的。

安裝 Anaconda 套件

裝他最重要的就是 (1) 基於課程要求 (2) 他包含大多數基本調程會用到的套件。

這裡我只說一下他的好處，就是他有瀏覽和搜尋的功能，能讓你快速的看出你裝了哪些Python的套件。我沒有再進一步的研究，在下載件的網頁上他也有問你要要不要申請會員，不是強制的，我就沒有申請了。

最後是，這次會用到的IDE環境 Spyder 在安裝 Anaconda 時也就一起裝好了。

Spyder::Python專用的IDE開發工具

Spyder is a free and open source scientific environment written in Python, for Python, and designed by and for scientists, engineers and data analysts.

像我這樣啥也不會的中年老白，一直聽到縮寫時，忍不住去科普一下自己啥是IDE。以下是WIKI的說明

整合開發環境（Integrated Development Environment，簡稱IDE，也稱為Integration Design Environment、Integration Debugging Environment）是一種輔助程式開發人員開發軟體的應用軟體，在開發工具內部就可以輔助編寫原始碼文字、並編譯打包成為可用的程式，有些甚至可以設計圖形介面。

簡單的說法是，他會幫你查看開發時的許多資訊，最常用的就是指令是什麼，指令怎麼用，以及跑執行的結果，協助你debug等等等。不裝也沒有不可以吧，但我是跟著課程學習，快最重要，這裡我就不多研究了。

到此為止，環境的安裝就算完成了。
