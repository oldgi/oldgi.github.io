上 Hawhow 課程：Python 入門特訓 - 基礎實作到證照攻略。 今天交作業。

作業 1 - 倍數的個數與總和

1_使用者輸入兩個正整數 a 和 b, 其中 a 必需 <= b 2_找到 a 和 b 之間 7 或 11 的倍數並印出 3_印出的時候，一列輸出 10 個數字，欄寬為 4，靠左對齊 4_最後印出倍數的個數及總和

下面是效果和我的代碼 

a=int(input("請輸入第一個整數 a : "))
b=int(input("請輸入第二個整數 b (請注意 b >= a): "))

while b < a:
    b=int(input("請輸入第二個整數 b (請注意 b >= a): "))
    
count=0
sum_num=0
i=a
while i <= b:    
    if i%7==0 or i%11==0:     
        count+=1        
        sum_num+=i
        if count%10 != 0:
            print("{:<4}".format(i),end="")
        else:
            print()
    i+=1
    
print("在{}和{}之間為 7 或 11 的位數 {} 個，加總為 {}".format(a,b,count,sum_num))

同樣的作業叫 chatGPT寫呢？

  • 這裡 count 用來計算輸了幾個數字。
  • 列印時想要以 ‘+’ 來串成輸出內容時，因為都要是 ‘str’ 型態才行，所以這裡有使用到像 str(count) 的寫法。

下面是效果和 chatGPT 的代碼，明顥比我簡潔又好讀多了。 

count=1
maxnum = int(input("請輸入第"+str(count)+"個數字，直到9999結束："))
userInput = maxnum
while userInput != 9999:
    if userInput > maxnum:
        maxnum = userInput
    count += 1
    userInput = int(input("請輸入第"+str(count)+"個數字，直到9999結束："))
print("輸入"+str(count)+"個數字，其中最大的值是 : ",maxnum)
