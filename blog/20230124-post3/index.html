上 Hawhow 課程：Python 入門特訓 - 基礎實作到證照攻略。 學程式語言，基本的語言型態、命名規則及內建關鍵字、型別轉換、及格式輸出，是這堂課認為最基本的東西。

這裡就不會一一列出課堂教的東西，在這幾個章節中，只把我自己寫出來的小作業貼上做為紀錄。

讓使用者輸入身高及體重後計算BMI並印出

課程裡有提到如何計算BMI的公式，這裡應用到的是：

  1. 在console產生提示語並接受一個輸入值。
  2. 將輸入值存到指定的變數。
  3. 使用除法與平方數學運算,並將結果存到提定的變數。
  4. 依指定的內容將資訊印出。

下面是我的代碼

# -*- coding: utf-8 -*-
"""
calculate BMI=weight / hight**2
"""

x=eval(input("請輸入身高(cm): "))
y=eval(input("請輸入體重(kg): "))
z=y/(x/100)**2
print("您的身高是: ",x,"    您的體重是: ",y,"    BMI計算結果為: ",z)

讓使用者輸入攝氏溫度後為他換算成華氏並印出

這裡應用到的是以 {} 將要印出的變數內容排版寫好，這樣在閱讀上也比較有結構。

# -*- coding: utf-8 -*-
"""
攝氏= 5/9 乘(華氏溫度- 32)。
華氏= (攝氏乘9/5) + 32 
"""

tempC=eval(input("請輸攝式溫度(C): "))
tempF=tempC*5/9+32

print("攝式溫度 ",tempC,"度相當於華式溫度 ",tempF,"度")
print("攝式溫度 {} 度，相當於華式溫度 {} 度".format(tempC, tempF))

# -*- coding: utf-8 -*-
"""
計算兩點之間的距離
p1=(x1,y1)
p2=(x2,y2)
distance=square((x1-x2)**2+(y1-y2)**2)
"""
x1=eval(input("請輪入p1的 X 座標 = "))
y1=eval(input("請輪入p1的 y 座標 = "))
x2=eval(input("請輪入p2的 X 座標 = "))
y2=eval(input("請輪入p2的 y 座標 = "))

distance="p1({},{}) 和 p2({},{}) 之間的距離為 {:10.3f}。".format(x1,y1,x2,y2,((x1-x2)**2+(y1-y2)**2)**0.5)

print(distance)
