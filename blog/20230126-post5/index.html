上 Hawhow 課程：Python 入門特訓 - 基礎實作到證照攻略。 學習使用 for & while 迴圏。

for 迴圏的小 tip：

  • 在巢狀迴圏時，動得比較快的數字要放內迴圏
  • 如果用來控制輸出排列要好看，又要固定數量換行（因為print預設結尾會帶’\n’），可以內層迴圏 print("{:<4}".format(i),end=' ') ，然後在上一層次迴圏要汞再做一次print()
  • 指定 for 迴圏的範圍時，使用的語法像是：for i in range(7):；其中的 i 是由 0~6；在使用時要特別注意。

while 迴圏的小 tip：

  • while 中的 break —— 有時候在某些條件成立之下，希望強制跳出一層的迴圈，可以使用 break 。
  • while 中的 continue —— 有時候當某個條件成立，是希望迴圈內的後面的程式在這一次不要執行（跳過之意)、但是要再繼續迴圈其他的重複次數；就會使用 continue。

這裡就不會一一列出課堂教的東西，一樣把課程上我寫的小作業貼上做為紀錄。

用 FOR 迴圏印出 4個方向的直角三角型

下面是效果和我的代碼 

n=eval(input("請輪入三角型階層數 : "))
print("直角(左下)三角型")
for i in range(1,n+1):
    print("＊"*i)

print("直角(右下)三角型")
for i in range(1,n+1):
    print("　"*(n-i)+"＊"*i)

print("直角(左上)三角型")
for i in range(1,n+1):
    print("＊"*(n-i+1))
    
print("直角(右上)三角型")
for i in range(n):
    print("　"*i+"＊"*(n-i))

加碼列印等腰三角型 下面是效果和我的代碼 

h=eval(input("請輪入等腰三角型的高 : "))

for i in range(h):
    print("　"*(h-i)+"＊"*(2*i+1))

輸入一堆數字、直到9999出現為止；從中找到最大值

  • 這裡 count 用來計算輸了幾個數字。
  • 列印時想要以 ‘+’ 來串成輸出內容時，因為都要是 ‘str’ 型態才行，所以這裡有使用到像 str(count) 的寫法。

下面是效果和我的代碼

count=1
maxnum = int(input("請輸入第"+str(count)+"個數字，直到9999結束："))
userInput = maxnum
while userInput != 9999:
    if userInput > maxnum:
        maxnum = userInput
    count += 1
    userInput = int(input("請輸入第"+str(count)+"個數字，直到9999結束："))
print("輸入"+str(count)+"個數字，其中最大的值是 : ",maxnum)
